Webhooks

▼ About webhooks
What they are

When something happens in Argus — a detection completes, a job finishes, a face is labeled — Argus sends an HTTP POST to a URL you configure. No polling required.

When to use one
  • Notify another service the moment a detection result is ready
  • React to async jobs finishing without polling /api/jobs/{id}
  • Trigger a downstream pipeline — indexing, tagging, alerts
Signing (optional)

Set a secret and every request will include X-Argus-Signature: sha256=<hmac-hex>. Verify it on your server: 

import hashlib, hmac
sig = request.headers["X-Argus-Signature"].removeprefix("sha256=")
expected = hmac.new(secret.encode(), request.body, hashlib.sha256).hexdigest()
if not hmac.compare_digest(sig, expected):
    raise ValueError("signature mismatch")
Setup
  1. Click + Add webhook above
  2. Enter the URL Argus will POST to
  3. Optionally add a label and a shared secret
  4. Check the events you want to receive
  5. Save, then click Test to verify connectivity
Events
detection.created

Fired after each detect call (synchronous, async, reprocess, bulk upload). Payload: source_image_id, external_ref, type. The type value reflects what was requested: "face", "object", or "all".

job.done

Fired when an async job completes or fails. On success the full result is merged into the payload. Single-image async jobs include source_image_id, external_ref, and detection arrays (faces/objects). Bulk upload jobs include total, type, and a results array (one entry per image). Payload always includes: job_id, status.

detection.labeled

Fired when a detection is assigned, reassigned, rejected, or unidentified (review queue, tag page, batch label, retroactive scan). Also fires when review-queue auto-confirm triggers. On reject or unidentify, identity_id and label are null. Payload: detection_id, source_image_id, identity_id, label, type.

identity.created

Fired when a new identity is created (via enroll, detect with a new label, or the identities API). Not fired when an existing identity is matched. Payload: identity_id, label, type, external_ref.

identity.updated

Fired when an identity is mutated: renamed, embedding added/removed, cover changed, or external_ref updated. Payload: identity_id, action ("renamed", "embedding_added", "embedding_removed", "thumbnail_updated", "external_ref_updated"). Rename also includes label and old_label; thumbnail_updated includes thumbnail_url; embedding_added/removed include detection_id (null when the embedding was deleted directly rather than via a detection); external_ref_updated includes external_ref.

identity.merged

Fired when one identity is merged into another (source is deleted). Payload: identity_id (deleted source), label, type, merged_into (surviving target id).

identity.deleted

Fired when a single identity is permanently deleted. Not fired for bulk delete-all. Payload: identity_id, label, type.

detection.deleted

Fired when detections are deleted: single deletion, bulk delete from the review queue, or source image deletion (which cascades to all its detections). Payload: detection_ids (array of deleted ids), count.

model.activated

Fired when an admin activates a new model. Broadcast to all subscribers across all environments — model changes affect every detection from that point on. Payload: model_id, name, type ("face" or "object").

Payload shapes
// detection.created
{ "event": "detection.created", "timestamp": "...",
  "data": { "source_image_id": 42, "external_ref": "photo-123", "type": "face" } }

// detection.labeled
{ "event": "detection.labeled", "timestamp": "...",
  "data": { "detection_id": 17, "source_image_id": 42,
            "identity_id": 5, "label": "Daisuke", "type": "face" } }

// identity.created
{ "event": "identity.created", "timestamp": "...",
  "data": { "identity_id": 5, "label": "Daisuke", "type": "face", "external_ref": null } }

// identity.updated (rename)
{ "event": "identity.updated", "timestamp": "...",
  "data": { "identity_id": 5, "label": "Daisuke Yamada",
            "old_label": "Daisuke", "action": "renamed" } }

// identity.updated (enrollment)
{ "event": "identity.updated", "timestamp": "...",
  "data": { "identity_id": 5, "action": "embedding_added", "detection_id": 17 } }

// identity.updated (unenroll)
{ "event": "identity.updated", "timestamp": "...",
  "data": { "identity_id": 5, "action": "embedding_removed", "detection_id": 17 } }

// identity.updated (embedding deleted directly — no associated detection)
{ "event": "identity.updated", "timestamp": "...",
  "data": { "identity_id": 5, "action": "embedding_removed", "detection_id": null } }

// identity.updated (cover changed)
{ "event": "identity.updated", "timestamp": "...",
  "data": { "identity_id": 5, "action": "thumbnail_updated",
            "thumbnail_url": "/media/crops/5/abc123.jpg" } }

// identity.updated (external_ref change)
{ "event": "identity.updated", "timestamp": "...",
  "data": { "identity_id": 5, "external_ref": "photo-abc", "action": "external_ref_updated" } }

// identity.merged  (source deleted, detections moved to target)
{ "event": "identity.merged", "timestamp": "...",
  "data": { "identity_id": 3, "label": "Daisuke (old)",
            "type": "face", "merged_into": 5 } }

// identity.deleted
{ "event": "identity.deleted", "timestamp": "...",
  "data": { "identity_id": 9, "label": "Unknown person", "type": "face" } }

// detection.deleted
{ "event": "detection.deleted", "timestamp": "...",
  "data": { "detection_ids": [14, 17, 23], "count": 3 } }

// model.activated
{ "event": "model.activated", "timestamp": "...",
  "data": { "model_id": 2, "name": "buffalo_l", "type": "face" } }

// job.done — single-image async (success)
{ "event": "job.done", "timestamp": "...",
  "data": { "job_id": "abc-123", "status": "done",
            "source_image_id": 42, "external_ref": "photo-123",
            "faces": [...], "objects": [...] } }

// job.done — bulk upload async (success)
{ "event": "job.done", "timestamp": "...",
  "data": { "job_id": "abc-456", "status": "done",
            "total": 5, "type": "face", "results": [...] } }

// job.done (failure)
{ "event": "job.done", "timestamp": "...",
  "data": { "job_id": "abc-123", "status": "failed", "error": "..." } }
